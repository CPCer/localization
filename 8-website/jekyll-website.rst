======================
使用Jekyll建站
======================

基础知识
================

使用Jekyll建站，需要掌握如下基本知识：
* Jekyll
* HTML，CSS，Javascript
* Liquid


Windows下安装Jekyll
-----------------------

Jekyll使用Ruby语言编写，可以参照官方文档 `Installation via RubyInstaller <https://jekyllrb.com/docs/windows/>`_中的步骤。简要步骤如下：

#. 下载 RubyInstaller，并安装
#. 运行 ``gem install jekyll bundler``
#. 检查是否安装成功 ``jekyll -v``
