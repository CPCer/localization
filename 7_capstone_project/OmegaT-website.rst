在本地化运行OmegaT网站
================================
在Mac Terminal中执行以下命令

#. 安装 html-proofer主题 ``gem install html-proofer``
#. 运行 ``bundle install`` 完成缺失 gems 的安装
#. 运行 bundle exec jekyll serve 以启动服务器

以上安装步骤，因为需要从服务器上下载文件，因为网络原因，安装时间或长或短，需要耐心等待。
