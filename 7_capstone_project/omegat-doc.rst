======================
OmegaT 文档
======================

语言资源识别
=================
omegaT的帮助文档是使用DocBook完成的


文件包的准备
======================
因为SDL Trados支持DocBook标准的xml，可以直接解析

#. 在 SDL Trados 中新建项目，并将所有xml文件导入项目
#. 使用打包功能将所有文件，翻译记忆库，术语库打成 sdlppx文件发送给译员



译后处理
================
#. 导入译员返回文件包
#. 生成译文xml


中文版帮助文档的编译（Win)
================================



#. 安装如下依存软件
    - DocBook XSL Stylesheets 1.75.2 ("dbk")
    - DocBook XML 4.5
    - fop 1.1
    - libxml2 2-2.7.7
    - Saxon 6-5-5
    - Ant 1.7.1 or above

#. 运行文件中的批任务 ``gendoc.bat``
